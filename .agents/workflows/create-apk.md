# Create Android APK Workflow

Follow these steps exactly to build a new Android APK for the mobile application.

1. Ensure all code is formatted and committed to git
// turbo
2. CD into the mobile directory and run `eas build -p android --profile preview` (or equivalent APK build command)
// turbo
3. Download the generated APK file
// turbo
